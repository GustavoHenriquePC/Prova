Clone: Fazer uma cópia idêntica de um repositório.
Commit: Cada commit registra as alterações que você fez nos arquivos.
Repositório: É um local onde seus arquivos e histórico de alterações são armazenados.
Merge: Combinar alterações de diferentes branches no Git.
Pull: Obter alterações do repositório remoto.
Branch:Uma linha de desenvolvimento separada
Pull request: Uma solicitação para mesclar suas alterações em um projeto
Fork: Fazer uma cópia de um projeto.
VCS: Software de controle de Versão
